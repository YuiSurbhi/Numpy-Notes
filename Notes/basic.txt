=> About Numpy :-

-> Numpy stands for (Numerical Python).
-> Numpy is a Python liberty which is commonly used in Numerical Computing.
-> This Python liberty is used for working arrays.
-> Numpy provide powerful N-Dimensional array object (ndarray), as well as supportive functions or tools used in these arrays.
-> It also has functions for working in domain of linear algebra, matrices etc.

=> History :-

-> Numpy was created by Travis Oliphant in 2005, while he was a graduate student at Mayo Clinic.

=> How can we use Numpy :-

-> import numpy as np
